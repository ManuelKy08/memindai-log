## Notice
skrip Perl yang menghasilkan log berupa permintaan HTTP untuk mencoba mengakses direktori atau file sensitif di server,
untuk mengidentifikasi potensi serangan pencarian direktori yang tidak aman. Setiap log yang dihasilkan 
mencatat alamat IP yang mencoba mengakses file, waktu akses, status HTTP (404 Not Found), dan informasi lain seperti 
user-agent yang menunjukkan bahwa permintaan tersebut berasal dari alat pemindai seperti Nmap. Direktori sensitif 
yang dicari termasuk file konfigurasi dan file tersembunyi yang berpotensi menyimpan informasi penting atau kerentanannya.
